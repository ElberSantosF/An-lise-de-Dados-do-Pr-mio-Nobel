Este projeto tem como objetivo realizar uma análise exploratória de dados (EDA) sobre o Prêmio Nobel, buscando identificar padrões históricos, tendências e insights relevantes relacionados aos laureados, categorias, países e distribuição ao longo do tempo.

O foco é demonstrar habilidades em análise de dados com Python, organização de raciocínio analítico e comunicação de resultados.
